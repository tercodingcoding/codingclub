CSS Selector

Selector di CSS digunakan untuk memilih elemen HTML yang ingin diberi style. Ada beberapa jenis selector yang sering digunakan:

  • Selector Tag: Memilih semua elemen dengan nama tag tertentu.
    p { color: red; } (semua paragraf jadi merah)
  • Selector Class: Memilih elemen dengan atribut class tertentu.
    .judul { font-weight: bold; } (semua elemen dengan class="judul")
  • Selector ID: Memilih elemen dengan atribut id tertentu.
    #utama { font-size: 30px; } (elemen dengan id="utama")
  • Selector Gabungan: Bisa menggabungkan beberapa selector.
    h1, h2 { color: blue; } (semua h1 dan h2 jadi biru)

Contoh:

<h1 class="judul" id="utama">Selamat Datang</h1>
<p>Ini paragraf biasa</p>
<p class="judul">Ini paragraf penting</p>

/* CSS: */
h1 { color: green; }
.judul { font-style: italic; }
#utama { text-decoration: underline; }

Catatan: Untuk latihan, silakan buka file latihan.html pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
