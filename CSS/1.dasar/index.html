CSS Dasar

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan layout halaman web. Dengan CSS, kamu bisa mengubah warna, ukuran, font, jarak, dan banyak lagi pada elemen HTML.

Cara Menyisipkan CSS

  1. Inline (langsung pada elemen HTML):
    <p style="color: red;">Teks ini merah</p>
  2. Internal (menggunakan tag <style> di <head>):
    <head>
  <style>
    p { color: blue; }
  </style>
</head>
  3. External (menggunakan file .css terpisah):
    <link rel="stylesheet" href="style.css">

Syntax CSS Dasar

selector {
  property: value;
}

Contoh:

h1 {
  color: green;
  font-size: 32px;
}

Artinya: Semua elemen <h1> akan berwarna hijau dan berukuran 32px.

Catatan: Untuk latihan, silakan buka file latihan.html pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
