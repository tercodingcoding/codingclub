HTML Table

Pada bagian ini, Anda akan belajar tentang penggunaan tag <table> untuk membuat tabel di HTML.

  • Materi & Contoh Table
    • <table> digunakan untuk membuat tabel di HTML.
    • <tr> (table row) untuk membuat baris pada tabel.
    • <th> (table header) untuk membuat judul kolom.
    • <td> (table data) untuk membuat sel data.
    • Setiap tabel minimal terdiri dari satu baris judul (<tr> berisi <th>) dan satu baris data (<tr> berisi <td>).
    • Contoh sederhana tabel: 
      <table border="1">
  <tr>
    <th>Nama</th>
    <th>Umur</th>
  </tr>
  <tr>
    <td>Ani</td>
    <td>20</td>
  </tr>
</table>
    • Hasil di browser:
    Nama Umur
    Ani 20

    Tips:

    • Gunakan atribut border agar tabel terlihat jelas saat belajar.
    • Anda bisa menambah baris data sebanyak yang dibutuhkan dengan menambah tag <tr> baru.
    • Untuk tabel yang lebih kompleks, bisa juga menggunakan <thead>, <tbody>, dan <tfoot> untuk membagi bagian tabel.

    Contoh penggunaan colspan dan rowspan:

    <table border="1">
  <tr>
    <th colspan="2">Data Mahasiswa</th>
  </tr>
  <tr>
    <td rowspan="2">Ani</td>
    <td>20</td>
  </tr>
  <tr>
    <td>Teknik Informatika</td>
  </tr>
</table>

    Hasil di browser:

    Data Mahasiswa
    Ani 20
    Teknik Informatika

    colspan digunakan untuk menggabungkan beberapa kolom, sedangkan rowspan untuk menggabungkan beberapa baris.

Catatan: Untuk latihan, silakan buka file latihan.html pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
