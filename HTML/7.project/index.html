Project: Mini Website Profil

Pada project ini, kamu akan membuat sebuah halaman profil sederhana yang menerapkan semua materi dari nomor 1 sampai 6:

  • Struktur HTML yang benar
  • Penggunaan heading untuk judul dan subjudul
  • Paragraf untuk deskripsi diri
  • Link (anchor) ke media sosial atau website favorit
  • Tabel untuk menampilkan data (misal: jadwal harian, pengalaman, atau skill)
  • Gambar (foto profil atau hobi)
  • List (ul/ol) untuk daftar hobi, skill, atau pencapaian

Context Project

Bayangkan kamu ingin memperkenalkan diri di internet. Buatlah halaman profil yang berisi:

  1. Judul utama: Nama kamu (gunakan <h1>)
  2. Foto profil (gunakan <img> dari internet atau lokal)
  3. Paragraf singkat tentang dirimu
  4. Daftar hobi atau skill (ul/ol)
  5. Tabel jadwal harian atau pengalaman (minimal 2 kolom, 2 baris)
  6. Link ke media sosial atau website favorit (minimal 2 link)

Catatan: Gunakan struktur HTML yang rapi dan sematkan komentar di setiap bagian untuk menunjukkan materi yang digunakan.

Untuk latihan, silakan buka file latihan.html pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
