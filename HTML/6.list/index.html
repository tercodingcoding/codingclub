HTML List

Pada bagian ini, Anda akan belajar tentang penggunaan tag <ul> (unordered list), <ol> (ordered list), dan <li> (list item) di HTML.

  • Materi & Contoh List
    • <ul> (unordered list) untuk membuat daftar tak berurutan (bullets).
    • <ol> (ordered list) untuk membuat daftar berurutan (angka/abjad).
    • <li> (list item) untuk setiap item di dalam list.
    • Contoh unordered list: 
      <ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
      • HTML
      • CSS
      • JavaScript
    • Contoh ordered list: 
      <ol>
  <li>Buka editor</li>
  <li>Tulis kode</li>
  <li>Simpan file</li>
</ol>
      1. Buka editor
      2. Tulis kode
      3. Simpan file
    • List bisa bersarang (nested list): 
      <ul>
  <li>Buah
    <ul>
      <li>Apel</li>
      <li>Jeruk</li>
    </ul>
  </li>
  <li>Sayur</li>
</ul>
      • Buah
        • Apel
        • Jeruk
      • Sayur
    • Ordered list bisa diubah tipenya dengan atribut type: 
      <ol type="A">
  <li>Satu</li>
  <li>Dua</li>
</ol>
      1. Satu
      2. Dua
    • Mulai nomor ordered list dari angka tertentu dengan atribut start: 
      <ol start="5">
  <li>Lima</li>
  <li>Enam</li>
</ol>
      1. Lima
      2. Enam

Catatan: Untuk latihan, silakan buka file latihan.html pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
