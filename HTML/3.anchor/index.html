HTML Anchor (Link)

Pada bagian ini, Anda akan belajar tentang penggunaan tag <a> untuk membuat link di HTML.

  • Materi & Contoh Anchor
    • <a> digunakan untuk membuat link di HTML.
    • Untuk membuka file dalam folder yang sama:
      <a href="latihan.html">Buka Latihan</a>
    • Untuk membuka file di folder berbeda:
      <a href="../1.headings/index.html">Ke Struktur HTML</a>
    • Untuk link ke website luar:
      <a href="https://www.example.com" target="_blank">Kunjungi Example</a>

    Contoh hasil di browser:

    Buka Latihan (folder sama)
    Ke Struktur HTML (folder berbeda)
    Kunjungi Example (website luar)

Catatan: Untuk latihan, silakan buka file latihan.html pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
