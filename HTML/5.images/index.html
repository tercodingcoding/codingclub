HTML Images

Pada bagian ini, Anda akan belajar tentang penggunaan tag <img> untuk menampilkan gambar di HTML.

  • Materi & Contoh Images
    • <img> digunakan untuk menampilkan gambar di HTML.
    • Wajib menggunakan atribut src (sumber gambar) dan alt (teks alternatif jika gambar gagal dimuat).
    • Bisa menambahkan atribut width dan height untuk mengatur ukuran gambar.
    • Contoh dasar: 
      <img src="https://www.w3schools.com/html/img_chania.jpg" alt="Pemandangan Kota Chania" width="200">
    • Hasil di browser:

    Tips:

    • Gunakan alt yang deskriptif agar website lebih ramah aksesibilitas dan SEO.
    • Gambar bisa dari file lokal (misal: src="gambar.jpg") atau dari internet (URL).
    • Untuk membuat gambar sebagai link, bungkus tag <img> dengan tag <a>: 
      <a href="https://www.example.com">
  <img src="logo.png" alt="Logo" width="100">
</a>
    • Bisa juga menambahkan title untuk tooltip saat mouse diarahkan ke gambar: 
      <img src="gambar.jpg" alt="Deskripsi" title="Ini tooltip gambar">

Catatan: Untuk latihan, silakan buka file latihan.html pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
