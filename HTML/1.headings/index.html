Struktur Dasar HTML

Pada bagian ini, kamu akan belajar tentang struktur dasar dokumen HTML, mengenal elemen, tag, dan bagaimana sebuah halaman web dibangun dari struktur tersebut.

  • Materi & Contoh Struktur HTML
    • Tag adalah penanda awal dan akhir elemen HTML, misal: <html>...</html>
    • Elemen adalah satu unit HTML yang terdiri dari tag pembuka, isi, dan tag penutup.
    • Struktur dasar HTML terdiri dari: <!DOCTYPE html>, <html>, <head>, <body>.

    Contoh kode struktur HTML:

    <!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama.</p>
  </body>
</html>

    Hasilnya di browser:

    Halo Dunia!

    Ini adalah paragraf pertama.

  • Materi & Contoh Heading
    • <h1> sampai <h6> adalah tag untuk judul dan subjudul.
    • <h1> untuk judul utama, <h2> untuk subjudul, dan seterusnya hingga <h6> untuk sub-bab paling kecil.

    Contoh kode heading:

    <h1>Judul Utama</h1>
<h2>Subjudul</h2>
<h3>Sub-subjudul</h3>

    Hasilnya di browser:

    Judul Utama

    Subjudul

    Sub-subjudul

Catatan: Untuk latihan, silakan buka file latihan1.html untuk latihan struktur HTML dan latihan2.html untuk latihan heading pada folder ini di code editor Anda, lalu kerjakan sesuai instruksi di dalam file tersebut.
